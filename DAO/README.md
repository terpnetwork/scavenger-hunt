# TestNET DAO 

TestNET DAO is an group of participants in the athena-1 test network of TerpNET.

TestNET DAO  will:

- Add members via governance voting 
- Vote on proposals for the TestNET Scavenger Hunt
- Boost community growth
- Organize TestNET & MainNET events

## Initial Members

These are the project/companies that are currently most active in the CosmWasm community.

- Discover Decentralization
- Terp Network community representatives 

More members can and will be added later.

Initially, decision making will happen by communication in open/closed channels such as TestNET DAO chats. Each member will provide an 
address which will be used for forming a 51% consensus. 

# Bounties

Bounties are well defined tasks with a reward attached. More info can be found on the [main bounties page](BOUNTIES.md).

# Proposals

Projects can ask for funds using proposals. You can view the full list of TestNET DAO Proposals (TDPs) [here](TDPs).

## Making a Proposal

Please use [TestNET DAO Proposal Template iwp-0](https://github.com/terpnetwork/scavenger-hunt/blob/main/DAO/TDPs/TDP-0.json) as basis 
for your proposal.

### Proposal Status

**Draft** - Initial ideation of a proposal.

**Review** - TestNET DAO Members marks an TDP as ready for and requesting Peer Review.

**Final** - Proposal finalized. All is delivered.

# Workgroups

For the internal and external projects, members of the TestNET DAO  can form workgroups and work on specific issues or topics.
