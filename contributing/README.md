## Test-Net Contribution Tracking: Athena-1

Terp Network is growing daily, and we want to be able to keep track those who are contributing to the growth of the network.

As described in this [repo](https://github.com/terpnetwork/scavenger-hunt), useful contributions during this migration process are tracked on a point based system, & will be used to propose & confirm allocations to active community members during the [fairdrop](https://docs.terp.network/overview/Fairdrop.html#goals-of-airdrop). 

## Types of Contributors

There are 3 ways to participate

1. Operate a node on Terp Network Test Network
2. Deploy or provide a Terp Network Dapp
3. Provide Useful Community Assistance During Phase 2 of Scavenger Hunt 

Each way of participation earns contributor one point, & will be publiclly tracked on a cvs sheet [here](/contributions.csv) through [github](https://github.com/terpnetwork), [discord](https://discord.gg/SYEjMwAac7), & [element](https://matrix.to/#/!MIEDknobAODITdWMZi:matrix.org?via=matrix.org).

*Feel free to push a PR to add your contributions within the github, & provide a link in a PR of this github highlighting your contribution!*


## Fairdrop Distribution 

Exact allocations to each contributor will be dependent on the total amount of contributors. the % of tokens allocated to participants will be confirmed via the Test-NET DAO. It is fair to note that all tokens distirbuting will be subject to a vesting schedule, up until **one year.**


It is encouraged to direct any questions regarding this process to the discord! Cheers.
